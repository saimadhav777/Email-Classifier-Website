About Our Platform

Welcome to our Email Classification website! This project was developed by Khundhan Bhargav and Sai Madhav, who are currently in their third year of B.Tech. Our aim is to create an efficient and user-friendly platform for classifying emails as either spam or not spam.

Our platform leverages state-of-the-art machine learning models to accurately detect and filter out spam emails. By using advanced natural language processing techniques, we ensure that only relevant and important emails make it to your inbox, while spam and phishing attempts are effectively blocked.

Technologies Used

We utilized a range of modern technologies to build this application:

  • Backend: Flask and MySQL
  • Frontend: HTML,CSS and JavaScript

Machine Learning Model

Our email classification model was trained using the following algorithms and processes:

  1. Data Preprocessing: Text transformation including tokenization, stemming, and removing stopwords using the NLTK library. Conversion of text data into numerical data using TfidfVectorizer.
  2. Model Training: We used the Multinomial Naive Bayes algorithm, a popular choice for text classification tasks.

Process Overview

  1. Data Collection: We gathered a labeled dataset containing spam and non-spam emails.
  2. Text Preprocessing: The text data was cleaned and preprocessed by converting it to lowercase, removing punctuation and stopwords, and applying stemming to standardize the words.
  3. Feature Extraction: We used TfidfVectorizer to convert the preprocessed text data into a matrix of TF-IDF features.
  4. Model Training: The Multinomial Naive Bayes model was trained on the TF-IDF features to learn patterns that distinguish spam emails from non-spam emails.
  5. Model Evaluation: The trained model was evaluated on a test dataset to ensure its accuracy and reliability.
  6. Deployment: The trained model and the vectorizer were saved using pickle and deployed using Flask, allowing users to interact with the model via a web interface.
